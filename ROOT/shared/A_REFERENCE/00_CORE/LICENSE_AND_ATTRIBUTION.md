## What this is

This is an **OPTIMIA-type structured memory** — a **5W2H bicameral memory architecture** (the "Generic
5W2H Memory Template V04"). It was created within **CEFCA's OPTIMIA project** (Centro de Estudios de Física
del Cosmos de Aragón) as a **generic, organisation-agnostic** template that any organisation can adopt and
adapt. It organises knowledge into a permanent reference model (`A_REFERENCE`), a live operations chamber
(`B_OPERATIONS`) and an operational kit (`C_OPERATIONAL_KIT`), governed by hard laws (canonical home +
read-only facets, roles-not-persons, human-in-the-loop, generated indexes, SEAL/WORM evidence, RAG-EXIT).

  OPTIMIA project.
