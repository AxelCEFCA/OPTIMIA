# _RESOURCES - brand & media assets (referenced, not embedded)

**What goes here.** The organisation's **logos**, images and visual assets. Working documents **reference** these (`<img src="../_RESOURCES/logos/primary.svg">`); they are embedded (base64) only when producing a heavy deliverable in `B_OPERATIONS/_OUTPUT`.

**What does NOT.** Generated heavy output (`B_OPERATIONS/_OUTPUT`), tools (`_TOOLS`), or content.

**Status.** Empty in the template. Drop the real assets here when configuring (CONFIG block B), and record their placement rules in `08_CORE_Profile`.
